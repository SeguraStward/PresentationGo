Instalación de Go

Guía Rápida de Instalación

Instalación en Windows

Pasos

  1. Visita https://go.dev/dl/
  2. Descarga el archivo MSI
  3. Ejecuta el instalador
  4. Abre PowerShell y verifica

Verificación

go version
go version go1.21.x windows/amd64

Instalación en Linux

# Descarga la última versión
wget https://go.dev/dl/go1.21.0.linux-amd64.tar.gz

# Extrae en /usr/local
sudo tar -C /usr/local -xzf go1.21.0.linux-amd64.tar.gz

# Agrega Go al PATH
echo 'export PATH=$PATH:/usr/local/go/bin' >> ~/.bashrc
source ~/.bashrc

# Verifica
go version

Instalación en macOS

Homebrew

brew install go
go version

Instalador PKG

  1. Visita https://go.dev/dl/
  2. Descarga el PKG para macOS
  3. Ejecuta el instalador
  4. Verifica con go version

Tu Primer Programa en Go

Crear Proyecto

mkdir hola-mundo
cd hola-mundo
go mod init hola-mundo

Archivo main.go

package main

import "fmt"

func main() {
    fmt.Println("¡Hola, Go!")
}

Ejecutar

# Ejecución directa
go run main.go

# O compilar
go build
./hola-mundo

Salida:

¡Hola, Go!
